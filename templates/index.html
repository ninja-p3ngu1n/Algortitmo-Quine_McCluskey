Calculadora y Tabla de Verdad usando Quine-McCluskey

Ejemplos de Álgebra Booleana:

Fáciles

  • A·B + C (A y B o C)
  • A'B (No A y B)
  • A + A'B (Absorción)

Difíciles

  • (A⊕B)·(C+D)' (XOR y NOR)
  • ~(A+B)C + AB'C'D (Múltiples negaciones)
  • A'B'C'D' + A'B'C'D + ABCD (Minitérminos)

Nota: Usa ' (comilla simple) justo después de una letra (Ej: A'). Puedes omitir el punto entre variables (Ej: AB es A·B). Para negar grupos entre paréntesis, usa el símbolo ~.

Haz clic en los ceros (0) de la columna "F" para encenderlos (1) y crear tu función.
