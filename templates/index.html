Quine-McCluskey

Minimización de Funciones Booleanas

Calculadora Booleana

Ejemplos Válidos:

  • A·B + C (A AND B OR C)
  • A'B o A'·B (NOT A AND B)
  • ~(A + B) (NOT (A OR B))
  • ABCD (A AND B AND C AND D)

Nota: Usa ' (comilla simple) justo después de una letra (Ej: A'). Puedes omitir el punto entre variables (Ej: AB es lo mismo que A·B). Para negar grupos entre paréntesis, usa el símbolo ~ (Ej: ~(A+B)).
